Conoce Javascript desde Cero

Un poco de historia

JavaScript (abreviado comúnmente "JS") es un lenguaje de programación interpretado, dialecto del estándar ECMAScript. Se define como orientado a objetos,3 basado en prototipos, imperativo, débilmente tipado y dinámico. JavaScript se diseñó con una sintaxis similar al C, aunque adopta nombres y convenciones del lenguaje de programación Java. Sin embargo Java y JavaScript no están relacionados y tienen semánticas y propósitos diferentes.

Array

¿Qué es un array?

Un array, lista o arreglo es un tipo de variable en el cual se pueden guardar varios tipos de datos usando el mismo nombre

Array en javascript

Un array en javascript se declara tal cual como si se tratara de una variable cambiando su contenido por para de corchetes >> [] en el cual situaremos nuestro array

Ejemplos de array en Javascript

var trabajadores = [];
trabajadores[0] = "Luis";
trabajadores[1] = "Fernando";
document.write(trabajadores[0]," ",trabajadores[1]);

La sentencia switch

La sentencia switch es usada comunmente cuando para una variable tenemos distintas respuestas, una forma de explicar esta sentencia seria la siguiente -> "La nomina de la empresa consta de jefes, supervisores y operadores, cuando marcan su hora de entrada cada tipo de trabajador recibe un mensaje y serie de instrucciones distintas"

Ejemplo de switch

var empleado = prompt("¿Qué tipo de empleado eres?");
switch(empleado){
case "jefe":
alert("Bienvenido Jefe");
break;
case "supervisor":
alert("Encantado de verlo Super");
break;
case "operador":
alert("A darle con todo campeón");
break;
default:
alert("No eres empleado");
break;

Búcles

¿Qué son Los búcles?

Un búcle en la programación es una sentencia que repite un pedazo de codigo mientras su condicional sea true, son comunmentes usados para recorre cadenas de texto, repetir un codigo cierta cantidad de veces y mucho más

El búcle for

el bucle for es una estructura de control encargada de determinar el numero de veces que una porción de codigo se ejecutara mientras su condicional se mantenga true, este búcle es implementado en nuestra vida cotidiana, un ejemplo de esto seria "Voy a trotar 10 km, cada vez que doy 1000 pasos, recorrí un 1km, debo repetir 10 veces, si ya pase los 10KM paro a descansar hasta otro día"

Ejemplo del búcle FOR

for(var pasos = 0; pasos <= 1000; pasos++){
document.write(pasos, "\n");
};

El búcle while

¿Qué es el bucle
while?

El búcle while es una estructura de control encargada de repetir infinitamente una sentencia hasta que la misma se convierta en false, explicar este búcle es sencillo --> "Pensemos en la regadera(ducha) de nuestro baño, si la abrimos el agua caera continuamente pase lo que pase y solo dejara de correr el agua si se cierra"

ejemplo del búcle de while

var ducha = 0;
while(ducha < 1000){
document.write("agua\n");
ducha++;
};

Podemos definir a una función como una gran variable encargada de guardar una porción de código que sera usada cierta cantidad de veces, explicar una función quedaria así --> "Una funcion en nuestra vida son las operaciones matematicas, la suma o la resta por ejemplo, ellas poseen un nombre, reciben parametros y ejecutan una serie de pasos"

Ejemplo de una
función

var resultado = a + b;
document.write(resultado);
};
